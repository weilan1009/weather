# weather 天气获取程序 # python 代码说明
**使用说明：**   
你需要先注册 OpenWeatherMap 账号，获取 API 密钥，并替换代码中的your_api_key_here  
运行程序后，按照提示输入城市名称进行查询  
输入q可以退出程序  

**功能特点：**  
查询全球任意城市的实时天气  
显示天气状况描述、温度、体感温度、湿度、风速等信息  
显示当地日出日落时间  
完整的错误处理机制  
友好的用户交互界面
